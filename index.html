This is an academic portfolio page in which you can find projects, mainly developed during my studies. I welcome you to navigate through the four main categories offered here. A short description of the concepts covered can be found under each category. I hope something catches your attention.

Categories

Machine Learning

Projects mainly deve­loped on my free time. There are four projects currently in development. One is a tomato disease classi­fier, a deep learning project using CNNs. The second is a pro­ject analy­zing cross sectional meteo­rological data for countries around the world. The dataset's time-range is along the past 40, or for some, the past 20 years. The third project is the predi­ction of the lumi­nocity value in a room, for which luminocity measure­ments are recorded every one second. Last but not least, a pro­ject, based on neural style trans­fer, mainly deve­loped as a hobby. Some of the results produced by the pre­viously mention­ed pro­ject can be found on my Insta­gram (link in footer).

Android

Various small projects of native android appli­cations, developed during my studies. The fea­tures cover layouts, acti­vities, forms, content pro­viders, photo galleries and inter­acting with the google maps API.

Web

Small scale projects mainly comprised of form-based appli­cations. I only develop the front end of these simple appli­cations, with Java­Script intera­cting with the local storage of the browser to store information added by the user. One of them is somewhat more complex, as it simu­lates a small­scale e-shop. It is the only appli­cation that involves back-end develop­ment.

Other

This is an IoT end-to-end system, developed for under­standing web-based tech­nologies on IoT Systems. The system contains both hard­ware and soft­ware compo­nents. It is a smart switch which can be switched on or off both auto­matically and over Wi-Fi (from the user's phone).

  There is also a stand­alone appli­cation, developed in python, for the needs of my free­lance job as a mixture classi­fier and msds author. The appl­cation inter­acts with a data­base that contains the pro­ducts and contains logic for classi­fying them.

Note: Currently, most projects are getting updated, with both impro­ving code (from the semester initially developed) and adding more featu­res. I would be extre­mely grateful for any feedback or tips on how to improve or change these projects. It would help immen­sely my develop­ment as a programmer.
